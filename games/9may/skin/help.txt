Для начала, несколько слов об управлении:

Во время игры:
  Квадрат     — скрыть/возвратить текст
  Крест       — перейти к следующей реплике
  L/R         - просмотр истории игры

В любой момент:
  Start       — вызвать/скрыть выпадающее меню
  Стрелки     — сменить активного элемента меню
  Крест       — выбрать активный элемент меню
  Треугольник — сделать скриншот в формате PNG

В разделе помощи:
  Стрелки,L/R - сменить страницу справки
  Select      - возврат в игру

----
Теперь, о самом движке:

Данный движок написан на языке Lua под платформу LuaPlayer. Т.е., это не полный порт RenPy-игры, а просто ее имитация.

В Lua-коде явно перечисляется, какие скрипты подгрузить. Если несколько из них имеют метку start, то выдается меню выбора.

В игре работает сохранение и загрузка через файл 'saves/default.sav', пока в открытом формате. Скриншот делается в файл 'screenshot.png', формат PNG.

В данный момент поддерживается простое следование, ветвление по вложенным меню, поддержка смены эмоций персонажа, переходы по меткам. Пока что нет звука, а смена фонов и спрайтов идет без эффектов, но все это планируется.

----
Полностью поддерживается RenPy-семантика:
  image SPRITE_ID [SPRITE_SUBID] = FILENAME
  scene SPRITE_ID
  show SPRITE_ID [SPRITE_SUBID] [at POSITION]
  hide SPRITE_ID
  label LABEL_ID:
  jump LABEL_ID
  [sound|music] play FILENAME
  [sound|music] stop
  $ код
  if код: elseif: elif: else:
  menu:
  #RANDOM_COMMENT_HERE
  return

Все остальное разбивается на лексемы, первая из которых может быть преобразована в имя говорящего.

Имена персонажей пока что вручную описываются в файле media.lua

----
Точный список поддерживаемых форматов изображений неизвестен. Движок совершенно точно может не поддерживать изображения большого размера, например 800х600. В демках используется JPEG и 8-битный PNG.

Для вывода латиницы и кириллицы используется черный моноширинный шрифт Lucida Console. Отображение посимвольное, кодировка текстов - 1251.
